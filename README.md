# PAC-MAN

Классическая аркадная игра «Пакман», реализованная на чистом JavaScript с использованием HTML5 Canvas и CSS3.

## 🚀 Особенности проекта
- **Ретро-дизайн:** Стилизация под игровые автоматы с неоновыми границами, глубоким черным фоном и пиксельным отображением (`image-rendering: pixelated`).
- **Динамическая анимация:** Пакман реалистично открывает и закрывает рот во время движения с помощью тригонометрических функций, меняя направление взгляда.
- **ИИ Призрака:** Противник вычисляет кратчайший путь к игроку, а при столкновении со стенами использует алгоритм случайного обхода.
- **Процедурный звук:** Звуковые эффекты поедания точек генерируются динамически через Web Audio API (`AudioContext`) без использования внешних тяжелых аудиофайлов.
- **Адаптивное управление:** Поддержка как классических стрелок, так и стандартной раскладки WASD.

## 🕹️ Управление
Используйте клавиатуру для смены направления движения Пакмана:
- **Вверх:** `Стрелка Вверх` или `W`
- **Вниз:** `Стрелка Вниз` или `S`
- **Влево:** `Стрелка Влево` или `A`
- **Вправо:** `Стрелка Вправо` или `D`

Для сброса прогресса и перезапуска раунда нажмите кнопку **«НОВАЯ ИГРА»**.

## 📁 Структура репозитория
- `index.html` — Разметка страницы, холст игрового поля (Canvas 448x496), счетчик очков и элементы управления.
- `style.css` — Стили оформления интерфейса, анимация кнопок и шрифты.
- `script.js` — Игровой движок: генерация карты по матрице, физика столкновения со стенами, логика начисления очков и цикл `gameLoop`.

## 🛠️ Запуск игры
1. Склонируйте или скачайте файлы репозитория в одну папку.
2. Откройте файл `index.html` в любом современном веб-браузере (Chrome, Firefox, Edge, Safari).
